Documentación del Proyecto: Seguidor de Línea

Introducción

Este proyecto consiste en un controlador de movimiento para un robot Seguidor de Línea de alta velocidad, diseñado para competencias de robótica. Utiliza un algoritmo de control PID (Proporcional, Integral y Derivativo) para mantener una trayectoria óptima.

Características Principales

  • Control PID: Algoritmo sintonizado mediante el método de Ziegler-Nichols.
  • FSM (Máquina de Estados): Gestión robusta de estados (STOP, ACEL, CONTROL).
  • Sensores QTR: Lectura reflectiva de alta precisión con calibración dinámica.
  • Configuración Flexible: Perfiles de corredor seleccionables (Nightfall, Argentum, Diego).

Configuración del Hardware

El sistema está basado en un microcontrolador ESP32, utilizando el periférico LEDC para el control PWM de los motores y timers de hardware para el tiempo de control.

Configuración del Proyecto (platformio.ini)

El comportamiento del robot se define mediante flags de compilación en el archivo de configuración:

  • Corredores disponibles: Permite seleccionar entre NIGHTFALL, ARGENTUM o DIEGO.
  • Modos de pista: Definición de LINEA_NEGRA o LINEA_BLANCA.
  • Debug: La bandera DEBUG habilita la salida por monitor serial.
  • Entornos de prueba: El proyecto incluye entornos para probar botones, lectura de sensores y control IR.
