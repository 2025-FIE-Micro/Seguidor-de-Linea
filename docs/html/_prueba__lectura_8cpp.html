Descripción detallada

Programa de prueba para la lectura y calibración de sensores QTR.

Este test permite verificar la correcta recepción de datos de los 8 sensores, comparando el modo de lectura (RC o Analógico) y visualizando la posición calculada de la línea a través del monitor serial.

Documentación de funciones

◆ loop()

void loop ( )

Lazo principal de muestreo.

Realiza la lectura de la línea cada 10ms. Si se usa RC, lee la línea negra; si se usa Analógico, lee la línea blanca. Imprime los valores brutos y la posición por serial.

◆ setup()

void setup ( )

Inicializa el puerto serial y configura el tipo de sensores y pines.

Ejecuta una rutina de calibración de 400 ciclos (aprox. 10 segundos) indicando el inicio y fin mediante el LED interno.
