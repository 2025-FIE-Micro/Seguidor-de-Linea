Descripción detallada

Programa de prueba para la gestión de interrupciones externas mediante botones.

Este test verifica la respuesta inmediata del hardware ante señales de entrada externas (botones) utilizando las funciones de interrupción de Arduino/ESP32 para conmutar un LED sin intervención del lazo principal.
