Descripción detallada

Cabecera para la gestión de las interrupciones externas (ISR). Este módulo maneja la lógica de inicio y detención de la rutina principal del robot a través de un pin de interrupción (ej. un botón o jumper de seguridad).

◆ setupInterrupciones()

void setupInterrupciones ( )

Configura el hardware de interrupciones y el temporizador.

Asocia los pines de los botones a sus ISR correspondientes y configura el Timer 0 del ESP32 con un prescaler de 80 para contar en microsegundos.

◆ timerInterrupcion()

void IRAM_ATTR timerInterrupcion ( )

ISR del temporizador de hardware.

Se ejecuta periódicamente para gestionar tareas de tiempo real, como el refresco del cálculo PID. Utiliza IRAM_ATTR para máxima velocidad de respuesta.

ISR del temporizador de hardware.

Solo señaliza que el tiempo ha expirado activando la bandera has_expired. Se ejecuta en IRAM para minimizar latencias.
