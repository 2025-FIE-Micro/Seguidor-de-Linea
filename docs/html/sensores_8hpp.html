Documentación de enumeraciones

◆ Linea

enum Linea

Define el modo de operación del seguidor de línea: Indica si el robot debe buscar la línea oscura (NEGRA) o la línea clara (BLANCA). Este valor se configura globalmente en config.cpp mediante la bandera LINEA_NEGRA.

Valores de enumeraciones
BLANCA 

El robot sigue la línea blanca sobre fondo negro (valores altos = sobre línea).

NEGRA 

El robot sigue la línea negra sobre fondo blanco (valores bajos = sobre línea).
